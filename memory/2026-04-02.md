### Lessons Learned
- Docker containers don't automatically pick up file edits — need docker cp + restart
- Weaviate v3 → v4 API migration: `client.data.object` → `client.data_object`
- CORS must include ALL domains (greenplot.ink AND www.greenplot.ink)
- OpenRouter model IDs don't use `openrouter/` prefix
- sessionStorage clears on tab close; localStorage persists across sessions
- NVIDIA embedding API returns 403 — use OpenRouter instead
- Patching Docker containers is temporary — need to commit to git for persistence
